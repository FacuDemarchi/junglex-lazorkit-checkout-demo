# Cómo enviar una transacción gasless en Devnet

1. Conecta tu wallet con passkey.
2. En la pantalla principal, pulsa “Transferir SOL (gasless)”.
3. La app crea una `Transaction` y usa `signAndSendTransaction`.
4. Al finalizar, verás el signature y un enlace al Explorer (Devnet).
5. Ajusta monto/destinatario en `WalletDemo.tsx` si lo necesitas.
